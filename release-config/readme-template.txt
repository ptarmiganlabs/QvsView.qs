Getting Started
---------------

1. Upload the included "qvsview-qs.zip" extension file to Qlik Sense:

   - Qlik Cloud:
     Open Management Console > Extensions > Add (+) > Upload the zip file.

   - Qlik Sense Enterprise on Windows (client-managed):
     Open QMC > Extensions > Import > Select the zip file.

2. Reload any open app editor pages in your browser.
   The extension will only appear in the assets panel after a page reload.

3. Drag the "QvsView.qs" visualization onto a sheet, then add a Dimension
   in the property panel pointing to the field that contains your script rows.
   The script will render with full syntax highlighting.
